北京行圆汽车信息技术有限公司成立于2016年12月，是致力于中国汽车产业的互联网平台服务供应商。行圆汽车分别为汽车生产企业及汽车经销商提供汽车车源批发服务，汽车互联网营销服务，经销商互联网运营服务。业务体系涵盖新车，二手车，汽车金融，汽车保险及汽车后市场等汽车全产业链。行圆汽车将通过运营旗下库存融资，媒体渠道，流量变现三大事业部，努力把公司打造成为最优秀的汽车产业互联网平台服务供应商和最具影响力的汽车消费者互联网服务供应商。

赤兔联盟

赤兔联盟是北京行圆汽车信息技术有限公司潜心研发的一站式智能分发营销平台。依托行圆汽车数十年汽车行业客户资源以及海量自媒体资源，依靠媒体智能匹配，内容全网分发，媒体智投三大服务体系，为汽车厂商、厂商大区和经销商提供全网域汽车整合营销解决方案，同时帮助媒体主完成其在汽车行业的流量变现
