奖励机制：

  • 从赤兔联盟平台将文章转发到社交圈，只要被有效点击浏览，转发者就会得到奖励；
  • 文章转发后请勿删除，保留时间越长、点击越多，则奖励越高。
  • 每个任务可以多次跨社交平台转发，能有效增加好友浏览奖励。
  • 每个用户每天仅计算前5条任务产生的收益，单篇文章分享收益上限为2元，其他视为免费分享。
  • 通过手机转发和点击浏览才能获得收益。

奖励计算规则：

  • 同一个粉丝/用户在同一篇文章分享有效期内仅计算一次有效点击。
  • 文章分享后2天内（含2天）产生的有效点击计算收益，超过2天不计入有效点击量，不计算奖励。
  • 每篇文章产生的收益于次日上午7:00以后在账户显示。
  • 由于平台问题导致的文章被删除或者任务下架，已经产生的收益会正常结算。
  • 有作弊嫌疑的点击，本次任务不计算收益，违规行为包括但不限于：机器作弊，刷IP，多人互点，诱导点击等

名词解释：

  • 社交圈：包括但不限于微信好友，微信群，朋友圈，微博，QQ空间，QQ好友，QQ群，今日头条，知乎，各大论坛等
  • CPC：根据用户有效点击次数计算收益。
  • 有效点击：同一个粉丝/用户在同一篇文章分享有效期内第一次点击视为有效点击。

关于提现：

  • 用户申请提现时，赤兔联盟平台会验证用户信息，如果信息不完善，需要完善信息后才可申请提现，平台会在您提交信息后24小时内审核资料。
  • 赤兔联盟平台会审核用户提现申请，审核通过后，会转账到用户绑定的支付宝账户，提现过程中，账户中的钱会处于提现中。
  • 对于疑似作弊账号进行排查确认，如果赤兔联盟确认该账号行为属于作弊，赤兔联盟将对该账号进行永久封号。
  • 从您提交提现申请，赤兔联盟将在7个工作日内为您处理，在此期间您的提现金额将一直处于“提现中”状态，如遇国家法定假日会顺延处理。
  • 满100元即可申请提现，同一用户24小时内仅可提现一次。
  • 不能提现情况：
  • 未完善用户资料，不能提现
  • 支付宝绑定了多个赤兔联盟账户，涉嫌作弊，不能提现
  • 支付宝账号填写错误或重复，不能提现
  • 支付宝账号未绑定身份证号码（需要报税），不能提现。
