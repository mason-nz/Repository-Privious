互动参数

  • * 直播观众数
  • 截止更新日期，所有直播的总直播观众数÷总直播次数,平均直播观众数越高，代表主播的人 气越高
  • * 最高观众数
  • 截止更新日期，所有直播的最高观众数
  • * 平均观众数
  • 截止更新日期，所有直播的总直播观众数÷总直播次数,平均直播观众数越高，代表主播的人 气越高
  • * 累计打赏数
  • 指账号累计收到的打赏数,例:映票、金币、花椒币、魅力值
  • * 累计收入
  • 截止更新日期，被打赏的总虚拟币换算为主播人民币收入 参照对应平台算法，主播收入计算如下(不考虑充值优惠及个别主播特殊分成): 映客:主播人民币收入=映票数/32; 花椒:主播人民币收入=花椒币*0.07; 一直播:主播人民币收入=金币数*0.003;
  • * 累计点赞数
  • 截止更新日期，主播的直播总点赞次数
  • * 累计送出数
  • 截止更新日期，主播在对应平台累计送出数
